## CoachRony — Personal Brand Platform

A premium, dark + glassmorphism, bilingual (Bangla + English) personal brand site for CoachRony with full backend (Lovable Cloud) for leads, bookings, courses, and a digital store. Phase-1 ships the complete marketing site + lead/booking + auth + store skeleton with placeholder content you can edit later.

---

### 1. Design system

- **Theme:** Dark base (near-black) + glassmorphism cards, purple/blue glow accents, animated gradient backgrounds.
- **Typography:** Poppins (display) + Inter (body), with Bangla fallback (Hind Siliguri) so Bangla copy renders cleanly.
- **Tokens (in `src/styles.css`, oklch):** background, foreground, primary (purple glow), accent (blue glow), card-glass, border-glow, gradient-primary, shadow-glow.
- **Motion:** framer-motion for hero, scroll-reveal on sections, hover lift on cards, animated gradient blobs.
